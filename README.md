# Vijay Kumar — 3D Portfolio

Interactive 3D portfolio website built with React, Three.js, and GSAP.

## Features

- 3D character model with mouse tracking
- Physics-based tech stack visualization
- GSAP scroll animations
- Custom cursor
- Responsive design

## Tech Stack

- React + TypeScript + Vite
- Three.js / React Three Fiber
- GSAP + ScrollTrigger
- Rapier Physics

## Getting Started

```bash
npm install
npm run dev
```
